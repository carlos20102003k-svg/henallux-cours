# henallux-cours
Ce dossier contient des fichiers et scripts utilisés **exclusivement dans le cadre d’un cours de forensics**.

Ils ont pour but d’illustrer des techniques d’analyse réseau, de tunneling SSH, de bruteforce, et de scan ICMP dans un environnement contrôlé.  
**Aucun de ces outils ne doit être utilisé en dehors d’un cadre légal et académique.**

⚠️ Toute utilisation abusive ou hors contexte pédagogique est strictement interdite.
